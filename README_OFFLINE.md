# サボる時計 v1.0.0（オフライン同梱版）

このフォルダは **外部ネットワーク（CDN / Google Fonts 等）に依存せず**動くように、
p5.js / フォント / MediaPipe（WASM + モデル）を同梱した展示用ビルドです。

## できること
- 顔（またはシミュレーションで「見られている」）を検知すると、登場演出が **毎回ランダム**で発火
  - **①-a スクスト**
  - **②-a スクスト + 一部おくれ**
- それ以外の演出（多方向ランダム／後ずさり／爆発／消滅など）は **削除・無効化**済み
- 「見られている（シミュレーション）」チェックと「カメラプレビューを表示」は残しています（デバッグ用）

## 起動方法（重要）
MediaPipe の ESM / WASM を読み込むため、**file:// 直開きでは動かない**場合があります。
**ローカルサーバー**で開いてください。

例：
- VSCode の Live Server
- macOS / Windows: `python -m http.server 8000` をこのフォルダで実行 → `http://localhost:8000/` を開く

## UI
- 左下の「≡」でメニューを開きます
- メニュー内：⛶ フルスクリーン / カメラプレビュー / 緑枠
- 「UIを隠す」はメニュー右上の「×」です

## 構成
- `index.html` : UI + 起動
- `src/main.js` : 本体ロジック
- `libs/p5.min.js` : p5.js
- `vendor/mediapipe/*` : MediaPipe（vision_bundle.mjs / wasm / model）
- `fonts/*` : Inter フォント
